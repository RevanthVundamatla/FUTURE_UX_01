# Website Usability Analysis & Redesign – Healthcare Clinic

This project presents a **Website Usability Analysis and UI/UX Redesign** of a healthcare clinic website.

The goal of this project was to analyze the usability of an existing website and redesign it with a modern, user-friendly interface that improves navigation, accessibility, and user experience.

## Project Overview

The original website had several usability issues such as unclear navigation, inconsistent layout, and poor visual hierarchy. After conducting a usability analysis, the website was redesigned to create a cleaner and more intuitive healthcare interface.

The redesigned UI focuses on improving the patient experience by providing clear navigation, structured content, and accessible healthcare information.

## Key Improvements

- Improved navigation structure
- Clean and modern UI design
- Better visual hierarchy
- Improved readability and spacing
- Clear call-to-action buttons
- Mobile-friendly design concept
